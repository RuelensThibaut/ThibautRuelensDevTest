Hello, Inimco!

This is the homepage for the Inimco Developer Test!

For this small project I chose the following technologies:

  • ASP .NET Core
  • Angular 8
  • Prime NG (Component Library)
  • Unit Test

On the User page you can see a button which will open a dialog. Here you can fill in the information and press Analyse. Once the analyses is done you will see the result on the page.
